Адекватного применения ServiceBroker для имеющейся части проекте я не нашел,
поэтому решил реализовать гипотетическую ситуацию - есть внешний источник данных для моей таблицы Employees 
- скрипт CreateTestDB_and_SBObjects.sql создает базу данных [TestServiceBrokerTransfer] со всеми объектами Service Broker, хранимыми процедурами и таблицей EmployeesSource, которую заполняет некоторыми данными
- в моем проекте в БД Iventory добавлены объекты Service Broker
- скрипт CheckWorkOfSB.sql показывает работу Service Broker:
    1) добавляем в Employees некоторые данные
    2) проверяем имя работника с ID = 3 в базе Iventory
    3) меняем имя работника c ID = 3 в базе [TestServiceBrokerTransfer]
    4) в базе [TestServiceBrokerTransfer] выполняем процедуру отправки сообщения через Service Broker
    5) ждем 5 секунд, чтобы в фоне отработала передача сообщения
    6) проверяем имя работника с ID = 3 в базе Iventory, если эффект презентации не словили, то оно должно было измениться =)
